
```
$ owncloud get <filename>
> get specified file
$ owncloud post <file> <dir>
> post specified file under <dir>
$ owncloud ls <dir>
> get file list under <dir>
$ owncloud mv <filepath> <dist>
> move/rename <filepath> file to <dist>
$ owncloud cp <file> <dist>
> copy <file> file to <dist>

$ owncloud mkdir <dirpath>
> make <dirpath> directory
$ owncloud rm <dir>
> remove <dir> directory
$ owncloud share get
> get all shared file (?)
$ owncloud share get <file>
> get <file>
$ owncloud share info <file>
> get infomation about shared <file>
$ owncloud share share <file> <shareType> <name>
> share <file> as <shareType> file with <name>
```
